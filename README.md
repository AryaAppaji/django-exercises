# django-exercises

This repository contains different examples implemented with django. Designed to help aspirants who want to step into django.

## Key Points

- This repository tries to stick with best industry ready practices
- This project uses `uv` as dependency manager.
- This Repository uses `Semantic Versioning(SemVer)` to document the changes happend in this reposiory.
- To track the changes please refer to [CHANGELOG.md](CHANGELOG.md)
